Enables bKash Payment Option on WooCommerce

== Description ==

This plugin enables bKash payment gateway on any website based on [WooCommerce](https://woocommerce.com/). It's been specially designed for Bangladeshi WooCommerce store owners to accept payments in Bangladeshi Taka with latest Alpha-Numeric Transaction ID support.

"bKash Payment Gateway" adds the following features:

* Enables shop owners to recieve payment holding a bKash Personal Account through "SEND MONEY".
* Agent Account holders also eligible to recieve payment using "CASH OUT" option.
* Gives shop owners option to apply bKash Transaction Fee on Cart Total.
* The latest Alpha-Numeric Transaction ID validation ready.
* Customizeable Payment Name, Instructions Field and Order Status.

Note:
- This is a WooCommerce based plugin, so WooCommerce must be installed and activated to use bKash Payment Gateway.
- You must have a active bKash account (Personal/Agent) to receive payments.

== Installation ==

Here is how you should install the plugin

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin.
3. Go to WooCommerce > Settings > Checkout > bKash and configure your bKash Payment Gateway settings.


== Frequently Asked Questions ==

= Does this plugin verify Transaction IDs? =

No, but it uses a special structure of the Transaction ID which helps to validate the valid format of TrxID.

= Does it support the latest Alpha-Numeric bKash Transaction ID? =

Yes, it supports the most recent form of bKash Transaction IDs.

= Can I recieve payment through my Merchant Account also? =

Sorry, it doesn't support the Merchant Account option at this moment. But you may get the option in a future release.
